What is a network Layer?
- This is where the connections occur, by sending packets of data 
back and forth between different networks. 
- The network layer is the 3rd in the OSI Model.
   - There is a high usage of routing, testing, and encryption 

If two people are in a different LAN(local access network) and they want to
send a message to each other, then their messages have to be addressed
and sent to Alice's network before it can reach the destination user's computer
    - This is all done under the network layer

What is a network?
    - A group of two or more connected computing devices, normally connected in a hub like a router.
    - Include things such as subnetworks, or smaller subdivisions of the network

What happens at the network layer?
    - Setting up the routes for data packets to take, checking to see if a server in another network is
    up and running, and addressing and recieving IP packets from other networks. 

What is a Packet?
    - All data sent over the Internet is broken down into smaller chunks called "packets"
    - A packet has a header, which contains information about the packet itself, and the body, which is
    actual data being sent 
    - Normally networking software attaches a header on each packet before it is sent to the internet, and
    the software can also understand how to handle the packet with the header. 

- A header contains information about the content, source, and destination of each packet.
    - An IP header contains the destination IP address of each packet
    - The total size of the packet
    - An indication of whether or not the packet has been fragmented in transit
    - A count of how many networks the packets has traaveled through.

What is an OSI(Open Systems Interconnection) model?
- A description of how the Internet works. Breaks down the functions involved in sending data over
  the Internet into 7 layers
    7. Application Layer
       * Data generated by and usable by software applications using the HTTP protocol. 
    6. Presentation Layer
       * Data is translated into a form the applcation can accept. HTTPS encryption and decryption
        happen here
    5. Session Layer
       * Control connections between computers(Also done at layer 4 by TCP Protocol)
    4. Transport Layer
       * Provides the means of transmitting data between the two connected parties, as well as controlling
        the quality of service. (TCP and UDP protocols used here)
    3. Network Layer
       * Handles the routing and sending of data between different networks.
       * Most important protocols on this layer are IP, IPsec, ICMP, IGMP, GRE.
    2. Data Link Layer
       * Communication between devices on the same network
       * Ethernet protocol used here
    1. Physical Layer
       * Packets are converted into electrical, radio, and optical pulses and transmitted into bits. 

What is a TCP/IP model?
- An alternative model of how the Internet works.
    4. Application Layer:
       * Layer 7 of the OSI model.
    3. Transport Layer
       * Layer 4 on the OSI model.
    2. Internet Layer
       * Layer 3 of the OSI model.
    1. Network Access Layer
       * Layers 1/2 of the OSI model.
